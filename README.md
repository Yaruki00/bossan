# Bossan

Bossan is high performance asynchronous Rack Web server.

## Installation

Add this line to your application's Gemfile:

    gem 'bossan'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bossan

## Requirements

bossan requires Ruby 1.9.

bossan supports linux only.
